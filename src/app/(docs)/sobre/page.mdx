# Sobre o projeto

A ideia do projeto é fazer um rede social para que os alunos dos cursos de tecnologia
tenham algum contato inicial com alguma linguagem de programação e um projeto real.

O projeto é open source pois assim é possível que qualquer pessoa possa contribuir com o desenvolvimento, seja implementando funcionalidades
corrigindo prováveis bugs, documentação, design, usabilidade, etc.

## Stack

Inicialmente a será focada para a web, a stack utilizada será uma stack fullstack utilizando Nextjs como framework, pela
facilidade de desenvolver tanto o front-end quanto o back-end na mesma aplicação.

Abaixo você pode ver uma lista do que estaremos utilizando:

- [React](https://react.dev/)
- [NextJS](https://nextjs.org/)
- [Typescript](https://www.typescriptlang.org/)
- [Postgres](https://www.postgresql.org/)
- [Vercel](https://vercel.com/home)
- [Github](https://github.com/social-uninter-cc)
